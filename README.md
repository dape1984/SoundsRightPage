# SoundsRight Official Pages

This repository hosts the public-facing website and compliance pages for the SoundsRight app. Deploy to any domain (e.g. GitHub Pages, Vercel, Netlify, or your own server).

## Pages

- **Home**: [index.html](index.html)
- **Product Features & Highlights**: [product.html](product.html)
- **Privacy Policy**: [privacy.html](privacy.html)
- **Support & Contact**: [support.html](support.html)

## Deployment

1. Upload the contents of this directory to your web server or static hosting.
2. All links use relative paths (`index.html`, `product.html`, etc.), so they work on any domain.
3. For App Store / Play Store:
   - **Privacy Policy URL**: `https://soundsright.cn/privacy.html`
   - **Support URL**: `https://soundsright.cn/support.html`
